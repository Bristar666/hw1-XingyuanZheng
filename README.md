# Homework 1 - Coding Assistant Onboarding

This repository contains my work for Homework 1. It shows how I used a coding assistant to create a draft file, then revised the final version myself and tracked the process with Git and GitHub.

## Files
- `assistant_draft.md`: the version created with the help of a coding assistant
- `final_version.md`: the version I revised myself
- `README.md`: a short summary of the project and workflow

## Git steps I used
I created the local project folder, initialized Git, added the files, made commits as I completed each step, connected the repository to GitHub, and pushed my changes to the remote repository.

Main commands I used:
- `git init`
- `git add .`
- `git commit -m "repository setup"`
- `git commit -m "assistant-created draft"`
- `git commit -m "my edited final version"`
- `git push`
